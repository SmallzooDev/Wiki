## 키 바인딩

- `<leader>gR`: LSP 참조 표시
- `<leader>gD`: 선언으로 이동
- `<leader>gd`: LSP 정의 표시
- `<leader>gi`: LSP 구현 표시
- `<leader>gt`: LSP 유형 정의 표시
- `<leader>ca`: 사용 가능한 코드 액션 표시
- `<leader>rn`: 스마트 리네임
- `<leader>D`: 버퍼 진단 표시
- `<leader>d`: 줄 진단 표시
- `<leader>[d`: 이전 진단으로 이동
- `<leader>]d`: 다음 진단으로 이동
- `<leader>K`: 커서 아래 항목에 대한 문서 표시
- `<leader>rs`: LSP 재시작
